🩺 AI-Powered Multiple Disease Prediction System

📌 Project Overview

This is a Machine Learning web application that predicts the risk of multiple diseases using trained ML models. The application provides predictions for Diabetes, Heart Disease, and Stroke through an easy-to-use Streamlit interface.

✨ Features

- 🩸 Diabetes Prediction
- ❤️ Heart Disease Prediction
- 🧠 Stroke Prediction
- 🌐 Interactive Streamlit Web App
- 🤖 Machine Learning Models
- 📊 Simple and User-Friendly Interface

🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Pickle
